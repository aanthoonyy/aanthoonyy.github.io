shrust

Rust Screenshot Tool
View on GitHub

Shrust is a Wayland-native screenshot CLI tool that I wrote in Rust one warm summer day. I wanted a screenshot utility that more closely resembled the Windows Screenshot Tool. Instead of relying on existing solutions, I combined preexisting tools (grim, slurp, wl-copy, and swappy) into a single cohesive tool called Shrust.
